sendメソッド

レシーバの持っているメソッドを呼び出すことができる

object.send(メソッドa, メソッドaの引数) 

```ruby
class User
	def hoge(x)
		puts "aaa"
	end
end

user = User.new

user.hoge

user.send(:hoge)
user.send("hoge")
```
